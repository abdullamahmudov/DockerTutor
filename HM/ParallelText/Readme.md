# Инстркция к запуску  
При запуске файла "ConsoleApp.exe" необходимо передать значения аргументов:  
|N|Описание|Пример|
| :----- | :----- | :----- |  
| 1 | Путь к папке с файлами | "D:\\HomeWorks\\Parallel\\Files" |  
| 2 | Лимит количества читаемых файлов | "3" |  

# Структура  

1. ConsoleApp - проект для запуска программы  
2. ParallelFileReader - проект для чтения файлов  
3. TextParser - проект для анализа файлов
