# Инстркция к запуску  
Неободимо запустить файл ConsoleApp.exe или ConsoleApp.dll  

## Структура  
### Ключевые интерфейсы   
1. ICalculator - классы реализуют методы вычисления соответсвующими способави  
2. ITester - классы реализуют методы запуска вычислений указанными способави и формируют результат  
3. IGenerator - классы реализуют методы генерирующие наборы/массивы чисел  
4. IInfo - классы реализуют методы получения информации (информации о системе)
