GitHub - `https://github.com/cline/kanban`

### Использовать версию node (22)
```bash
nvm use 22
```

### Установка версии node по умолчанию
```bash
nvm alias default 22
```

### Версия node
```bash
node -v
```

### Список node/настройки nvm
```bash
nvm ls
```

### Установка версии node
```bash
nvm install 22
```

### Путь к node
```bash
which node
```

### Сборка kanban
```bash
npx kanban
```

### Запуск kanban
```bash
kanban
```

### Запуск kanban (Or install globally)
```bash
npm i -g kanban
kanban
```
